## 🚀 **Getting Started**

### **For Users**
1. **Connect Wallet**: Use Valora, MetaMask, or any WalletConnect wallet
2. **Complete Onboarding**: AI-guided profile creation with personality insights
3. **Start Matching**: Swipe through AI-curated profiles with compatibility scores
4. **Engage & Tip**: Super like with cUSD tips to show genuine interest
5. **Upgrade Experience**: Subscribe to premium tiers for enhanced features

### **For Developers**

#### **Frontend Setup**
```bash
cd front-end
npm install
npm run dev
```

#### **Smart Contract Deployment**
```bash
cd smartcontracts
npm install
npm run deploy:mainnet  # Deploy to Celo Mainnet
npm run verify:mainnet  # Verify on Celoscan
```
